# Governance Charter

This repository publishes a standards-style specification for Execution-Time Governance (ETG).

## Goals
- define interoperable artifacts for execution-time governance
- provide deterministic verification rules
- publish test vectors and a minimal reference verifier

## Non-goals
- standardize business policy logic
- standardize rail integrations
- publish proprietary enforcement orchestration

## Stewardship
CryptoCardia is the initial steward of ETG v1. The steward:
- accepts issues and pull requests
- publishes tagged releases for stable versions
- maintains registries for algorithms and canonicalization rules
